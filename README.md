# Club Deportivo Latino — Landing Page

This repository contains the source code for the **Club Deportivo Latino** landing page. It represents the first public version of the club’s website, designed to introduce the organization, present key information, and provide visitors with a simple way to get in touch.

## 🚀 Tech Stack

This project is built using modern web technologies:

* **Next.js**
* **React**
* **TypeScript**
* **JavaScript**
* **CSS**
* **TailwindCSS**
* **Next-Intl**

* **Cloudinary** (for media handling)

## 📝 About This Version

This initial release focuses on providing:

* A clean and responsive **landing page**
* Basic information about the **football club**
* **Contact forms** integrated through JotForm
* A lightweight and performant architecture

## 📌 Features

* Static and dynamic sections built with Next.js routing
* Type-safe components developed in TypeScript
* Media assets hosted and optimized via Cloudinary
* TailwindCSS styling for rapid and consistent UI development

## 📦 Installation & Setup

```bash
# Install dependencies
yarn install
# or
npm install

# Run development server
yarn dev
# or
npm run dev
```

Open your browser at **[https://deportivo-latino.vercel.app](https://deportivo-latino.vercel.app)**.
